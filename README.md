MxIPTV player is player for Android TV / Google TV

For TVs and boxes only, touch UI is not supported

The application supports xtream codes and m3u playlists

The playlist is sent to the application using the form on the website https://mxpl.eu/

Languages: English, Czech, Slovak, Polski, Magyar, Deutsch, Espanol, Italiano, Russ, Francais, Arab..

The application does not contain any playlists or other content.
